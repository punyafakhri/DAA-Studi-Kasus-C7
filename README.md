# DAA Studi Kasus C7 - Travelling Salesman Problem

Repositori ini berisi implementasi program dalam bahasa **Java** untuk menyelesaikan optimasi rute **Travelling Salesman Problem (TSP)** menggunakan pendekatan **Algoritma Brute Force (Exhaustive Search)**.

Tugas Studi Kasus C7 ini dikerjakan untuk memenuhi evaluasi mata kuliah Desain dan Analisis Algoritma (DAA) di Fakultas Ilmu Komputer, Universitas Brawijaya.

## 🗂️ Struktur Repositori
* `TSPBruteForce.java` : *Source code* utama implementasi algoritma pencarian permutasi rute.
* `Diagram Pohon Rekursi.png` : Ilustrasi visual kedalaman memori (*Call Stack*) untuk pembuktian kompleksitas ruang $O(n)$.
* `Studi Kasus C7_Muhammad Fakhri Aldiansyah_245150201111049 DAA TIF C.pdf` : Dokumen laporan lengkap yang berisi pseudocode, analisis matematis kompleksitas waktu $O(n!)$, dan kompleksitas ruang $O(n)$.

## 🚀 Cara Menjalankan Program
1. Pastikan Java Development Kit (JDK) sudah terinstal di komputer.
2. *Clone* repositori ini atau *download file* `TSPBruteForce.java`.
3. Buka terminal atau *command prompt*, lalu navigasi ke direktori *file*.
4. *Compile* program dengan perintah: `javac TSPBruteForce.java`
5. Jalankan program dengan perintah: `java TSPBruteForce`
