# DAA-Studi-Kasus-C7

Repositori ini berisi implementasi *source code* dalam bahasa Java untuk menyelesaikan *Travelling Salesman Problem* (TSP) menggunakan pendekatan Algoritma Brute Force (*Exhaustive Search*). 

Studi kasus ini dikerjakan untuk memenuhi tugas mata kuliah Desain dan Analisis Algoritma (DAA) semester 4.

## Penjelasan Kode
Program ini akan mengevaluasi seluruh permutasi rute perjalanan dari kota awal (kota 0) ke n-1 kota lainnya, menghitung total jarak setiap rute, dan mengembalikan rute optimal dengan jarak paling minimum.
